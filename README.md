#Install & Run 
<p>Download the application</p>
<pre>
  <code>$ git clone git@github.com:geektech893916/mean-stack-contact-manager.git</code>
</pre>
<p>Install</p>
<pre>
  <code>
    $ cd mean-stack-contact-manager
    $ npm install
  </code>
</pre>
<p>Run</p>
<pre>
  <code>$ node server.js</code>
</pre>
